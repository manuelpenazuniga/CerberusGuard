# CerberusGuard Dashboard

Live event feed for the CerberusGuard collector.

## Quickstart

1. Ensure collector is running on `http://localhost:9090`.
2. Install dependencies:

```bash
npm install
```

3. Start dev server:

```bash
npm run dev
```

4. Open `http://localhost:3000`.

## Build

```bash
npm run build
```
